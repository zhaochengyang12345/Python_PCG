# PCG 心音杂音孤立数据集处理工具 — 技术报告

---

## 1 项目背景与目标

心音图（Phonocardiogram, PCG）是记录心脏机械振动的无创信号，包含 S1（第一心音）、收缩期、S2（第二心音）、舒张期四个时段。公开数据集（如 PhysioNet 2016、CirCor DigiScope 2022）通常以**标注好边界的短片段**形式提供，而自建采集数据往往是数十秒乃至数分钟的连续长录音，且缺乏逐样本的状态标注。

本工具的目标是**将自建 WAV/CSV 格式 PCG 长录音处理成与公开数据集风格一致的标注短片段**，处理过程分为三个递进步骤，每步聚焦不同层级的信息提取：

### Step 1 — 心动周期切段

**核心目标**：将无时间标注的长录音切割成具有生理完整性的短片段。

公开数据集中每条记录通常对应 2–6 个完整心动周期（约 2–5 秒），是下游机器学习模型训练的基本单位。自建长录音直接使用不但时长不匹配，还包含录音头尾的噪声和体位移动伪影。Step 1 借助 Springer HSMM 模型识别每个 S1 起始位置，以此为边界贪心地划分段落，每段保证包含完整的 S1→收缩期→S2→舒张期循环，输出与公开数据集结构兼容的带标注短片段。

### Step 2 — 疾病掩膜（含 S1/S2 边界上下文）

**核心目标**：根据疾病病理生理学，屏蔽与该疾病无关的心动时段，突显杂音所在期相。

不同疾病的杂音有固定的发生时期——收缩期杂音（AS、MR、MVP）出现在 S1 之后、S2 之前，舒张期杂音（AR、MS）出现在 S2 之后、下一个 S1 之前。将无关时段置零后，模型仅需关注有信息量的区段，可提升分类精度、降低误判。

Step 2 **有意保留 S1 和 S2**，因为 S1/S2 是杂音定位的时相锚点——舒张期杂音必须在 S2 之后才能识别，收缩期杂音的"递增递减"形态必须以 S1 为起点才能量化。保留边界心音使研究者和模型均能判断杂音与心音的时间关系。

### Step 3 — 纯杂音提取

**核心目标**：在 Step 2 基础上进一步剔除 S1 和 S2 本体，获得仅含杂音成分的纯净音频。

S1（低频，20–100 Hz）和 S2（略高频，50–150 Hz）本身能量较强，若保留会干扰频谱特征的提取。在已知 S1/S2 精确边界的前提下，将其置零可得到"孤立杂音"段，便于：① 提取杂音时频特征（MFCC、Mel 谱等）用于分类；② 与公开数据集中已人工剔除心音的孤立杂音片段直接对比；③ 实现跨数据集迁移学习时统一输入格式。

---

三步处理总览：

| 步骤 | 功能 | 输出信号特征 |
|------|------|-------------|
| Step 1 | 长录音 → 2–6 周期短片段，带逐样本状态标注 | 完整 PCG 波形，含全部四个时段 |
| Step 2 | 按疾病屏蔽无关时段，保留杂音期相及 S1/S2 边界 | 非杂音区静默，S1/S2 与杂音区保留 |
| Step 3 | 进一步剔除 S1/S2，仅保留纯杂音 | 仅有纯杂音，心音本体已置零 |

## 3 信号处理流程

### 3.1 信号加载与预处理

**支持格式**：
- **WAV 文件**：固定采样率 4000 Hz，若不符则多相重采样至 4000 Hz
- **CSV 文件**：读取 `pcg` 列，自动从时间列推断采样率（默认 8000 Hz），对缺失值（NaN）使用 PCHIP 插值填补

**显示预处理链**（所有可视化波形均经过以下步骤，确保杂音形态真实还原）：

```
原始信号
  → Hampel 滤波         （窗长 40 点，σ = 2.5，去除脉冲伪影）
  → 陷波滤波            （50 Hz / 100 Hz，消除工频干扰）
  → Butterworth 带通    （20–400 Hz，4 阶，保留杂音频段）
  → 幅度归一化          （99 百分位数裁剪，[-1, 1]）
```

**Springer HSMM 状态定义**（分割结果，贯穿全流程）：

| 状态编号 | 含义 | 可视化颜色 |
|----------|------|-----------|
| 1 | S1（第一心音） | 玫瑰红 #e05c7a |
| 2 | 收缩期（Systole） | 橙色 #fab387 |
| 3 | S2（第二心音） | 琥珀色 #f5a623 |
| 4 | 舒张期（Diastole） | 蓝色 #89b4fa |

模型训练使用 `springer_hsmm/data/example_data.mat` 中前 5 条记录（`band_pi_matrices` 硬性下限），内部处理采样率统一降至 1000 Hz，特征帧率 50 Hz。模型为全局单例，使用线程锁保证后台线程安全调用。

---

### 3.2 Step 1 — 心动周期切段

**流程**：

```
长录音 (WAV/CSV)
  → Springer HSMM 推断逐样本状态序列 (1/2/3/4)
  → 检测 S1 上升沿（由非1→1 的跳变时刻）得到所有心动周期起始点
  → 贪心合并：尽可能凑满 max_cycles 个周期，不足时退而取 min_cycles
  → 输出多个短片段，每段以 S1 起始、以下一个 S1 作为右边界
```

每段输出字段：

```
signal        — 原始片段采样数组
states        — 逐样本状态标注（1/2/3/4），与 signal 等长
fs            — 采样率
cycle_count   — 包含的完整心动周期数（2–6）
start_sample  — 在原录音中的起始样本索引
end_sample    — 在原录音中的结束样本索引
```

**边界条件处理**：若信号末尾剩余心动周期数不足 `min_cycles`，该段丢弃，避免产生不完整片段。

---

### 3.3 Step 2 — 疾病掩膜（含 S1/S2 边界）

**流程**：

```
Step 1 片段 (signal + states)
  → 从文件名/目录名关键字自动识别疾病代码（AS/AR/MR/MS/MVP/N）
  → 查询内建疾病-保留状态映射表
  → 非关注状态的采样点置零 → masked_signal
```

**疾病-保留状态映射表**：

| 疾病代码 | 疾病名称 | 杂音期相 | 保留状态 | 典型频率范围 |
|----------|----------|----------|---------|-------------|
| AS | 主动脉瓣狭窄 | 收缩期（喷射型） | S1 + 收缩期 + S2 | 100–400 Hz |
| AR | 主动脉瓣反流 | 舒张期（递减型） | S1 + S2 + 舒张期 | 200–500 Hz |
| MR | 二尖瓣反流 | 收缩期（全收缩期） | S1 + 收缩期 + S2 | 150–400 Hz |
| MS | 二尖瓣狭窄 | 舒张期（隆隆样） | S1 + S2 + 舒张期 | 30–120 Hz |
| MVP | 二尖瓣脱垂 | 收缩期（喀喇音+渐增型） | S1 + 收缩期 + S2 | 100–400 Hz |
| N | 正常 | 无 | 全部保留 | — |

> 收缩期疾病（AS/MR/MVP）保留 **S1 + 收缩期 + S2**，舒张期疾病（AR/MS）保留 **S1 + S2 + 舒张期**。两类疾病均保留 S1 和 S2，原因在于 S1/S2 是杂音的时相定位锚点，缺失任一会导致杂音与正常心动周期的时间关系无法判断。

疾病标签通过文件名或父目录名中的关键字大小写不敏感匹配，用户亦可通过 GUI 手动覆盖。

**全疾病额外心音（S3/S4）处理策略**：

Springer HSMM 将 S3/S4 归入状态 4（舒张期），无法独立输出其标签。各疾病的处理方式取决于两个因素：① 该疾病是否可能出现 S3/S4；② 默认掩膜是否已保留舒张期。

| 疾病 | 可能的额外心音 | 出现率 | 默认保留状态 | 舒张期默认保留？ | S3/S4 处理方式 |
|------|-------------|--------|------------|:--------------:|---------------|
| AS | S4（舒张晚期，S1 前约 100 ms，20–70 Hz） | 重度约 30–40% | S1+收缩期+S2 | **否** | DWT 辅助提示；确认后手动勾选舒张期 |
| AR | S3（舒张早期，S2 后 120–180 ms，20–50 Hz） | 重度/急性可有 | S1+S2+舒张期 | **是** | 随舒张期**自动包含**，无需额外操作 |
| MR | S3（舒张早期，S2 后 120–180 ms，20–50 Hz） | 中重度约 40–60% | S1+收缩期+S2 | **否** | DWT 辅助提示；确认后手动勾选舒张期 |
| MS | OS 开瓣音（S2 后 60–120 ms，80–250 Hz）† | 几乎必然存在 | S1+S2+舒张期 | **是** | 随舒张期**自动包含** |
| MVP | 无 S3/S4 | — | S1+收缩期+S2 | 否 | 不适用 |
| N | 无 | — | 全部 | 是 | 全保留 |

> † MS 的开瓣音（Opening Snap）是二尖瓣弹性开放产生的高频短促音（80–250 Hz），与 S3/S4 性质不同（S3/S4 为低频充盈音，20–50 Hz），但均发生在舒张早期，随默认掩膜的舒张期保留一并纳入。

**对于 AS/MR（S3/S4 偶发且舒张期默认不保留）的辅助判断流程**：

若无条件为所有 AS/MR 录音保留舒张期，会在无额外心音的片段中引入无关静默段污染数据集。因此工具采用「默认关闭 + DWT 辅助提示」策略：界面自动对当前片段运行 `detect_extra_sounds()` 检测（详见 3.5 节），在波形子图 1 上用**紫色**标注 S3 候选、**青色**标注 S4 候选；用户据此结合临床经验在手动模式中决定是否启用舒张期保留。

### 3.4 Step 3 — 纯杂音提取

**流程**：

```
Step 2 掩膜信号 (masked_signal + states)
  → 将 states == 1（S1）的采样点置零
  → 将 states == 3（S2）的采样点置零
  → 输出 murmur_only_signal（仅含收缩期或舒张期杂音）
```

Step 3 以 Step 2 的输出为输入，因此：
- 收缩期疾病（AS/MR/MVP）：结果仅含收缩期杂音段
- 舒张期疾病（AR/MS）：结果仅含舒张期杂音段
- 正常（N）：结果保留收缩期 + 舒张期

此输出与公开数据集中"孤立杂音"片段的格式完全对齐，可直接用于特征提取（MFCC、Mel 谱）或跨数据集迁移学习。

---

### 3.5 额外心音辅助检测（DWT）

**背景**：Springer HSMM 的 4 状态模型无法区分舒张期内是否存在 S3/S4。如 3.3 节所述，AR 与 MS 的舒张期默认保留，S3/开瓣音会随之自动包含；而 **AS（可伴 S4）与 MR（可伴 S3）的舒张期默认不保留**，需要工具给出辅助线索供用户决策。`detect_extra_sounds()` 针对上述场景，在 Step 1 完成分割后对每个片段运行基于离散小波变换（DWT）的低频能量检测，结果仅用于**波形可视化提示**，不改变 Step 2/3 的掩膜逻辑，最终决策权在用户手中。

**算法流程**：

```
原始信号
  → pywt.wavedec(signal, "db4", level=N)
        ↓ 分解为多尺度细节系数层
  → 自动选取覆盖 20–70 Hz 的细节层（随 fs 自动计算）
        fs=4000: 主选 level 6（对应 31–62 Hz）
        fs=8000: 主选 level 7（对应 31–62 Hz）
  → 其余层系数置零 → pywt.waverec 重建低频分量 sig_wavelet
  → 搜索 S3 窗口（S2 结束后 80–220 ms）：
        目标窗小波均方能量 ÷ 舒张中段参考能量 ≥ 阈值 → S3 候选
  → 搜索 S4 窗口（S1 开始前 50–150 ms）：
        目标窗小波均方能量 ÷ 前心动周期舒张中段参考能量 ≥ 阈值 → S4 候选
```

**DWT 层级选择依据**：db4 小波（Daubechies 4 阶）与心音瞬态波形形态匹配良好；层级 $d$ 对应频率区间 $[f_s/2^{d+1},\; f_s/2^d]$，选取与目标频段 20–70 Hz 有交集的全部层级，频率覆盖随采样率自适应。

**参考能量区间**：选取同心动周期舒张期中段（S2 结束后 300–600 ms）作为本底噪声参考，避免将舒张晚期残余杂音或提前出现的 S1 纳入基线。

**GUI 展示**：
- S3 候选：**紫色**高亮区块（`#a855f7`），标注 `S3? (Nx)` 能量比
- S4 候选：**青色**高亮区块（`#06b6d4`），标注 `S4? (Nx)` 能量比
- 子图 1 标题附加 `⚠ 检测到 N 处 S3/S4 候选` 提示

> **注意**：该检测为启发式规则，低频段信号易受体位噪声干扰，存在假阳性。用户应结合临床判断决定是否在手动模式中勾选舒张期保留以捕获额外心音。

### 4.3 手动分段编辑器（SegEditorDialog）

|------|------|

批量保存产物：
1. **音频文件**：16-bit WAV，保持原始采样率，命名规则为：
   ```
   {源文件名}_{疾病代码}_{步骤后缀}_{序号:03d}_{周期数}cyc.wav
   ```
   其中步骤后缀：Step1 = `seg`、Step2 = `masked`、Step3 = `murmur`

   ```json
   {
     "index": 1,
     "source_file": "D:/data/AS/patient001.wav",
     "disease": "AS",
     "cycle_count": 3,
     "fs": 4000,
     "duration_s": 2.415,
     "step": 3
   }
   ```

## 6 疾病生理学依据

本节补充 Step 2 掩膜区段设计的病理生理学依据，为 3.3 节中的映射表提供临床支撑。

**收缩期杂音疾病（AS / MR / MVP）→ Step 2 保留：S1 + 收缩期 + S2**

- **AS（主动脉瓣狭窄）**：主动脉瓣开放受阻，血流在收缩期向主动脉射出时因流速加快产生喷射型递增递减杂音（S1 后→S2 前），频率 100–400 Hz。重度时可伴射血喀喇音。约 30–40% 的重度 AS 病例存在 **S4**（舒张晚期低频音，S1 前约 100 ms，20–70 Hz），提示左室顺应性下降。S4 位于舒张期（状态 4），默认掩膜不保留；若 DWT 检测器（3.5 节）标注 S4 候选且临床确认，可手动启用舒张期保留。
- **MR（二尖瓣反流）**：二尖瓣关闭不全，收缩期左室压力上升时血流经关闭不全的瓣口反流入左房，产生贯穿整个收缩期的吹风样高调杂音（150–400 Hz）。约 40–60% 的中重度 MR 病例可闻及 **S3**（舒张早期低频充盈音，S2 后约 120–180 ms，20–50 Hz），反映左室容量负荷过重。S3 位于舒张期（状态 4），默认掩膜不保留；若 DWT 检测器（3.5 节）标注 S3 候选且临床确认，可手动启用舒张期保留。
- **MVP（二尖瓣脱垂）**：收缩中期二尖瓣叶脱入左房时发出喀喇音（100–300 Hz），之后产生收缩晚期渐增型吹风样杂音（150–400 Hz）。

**舒张期杂音疾病（AR / MS）→ Step 2 保留：S1 + S2 + 舒张期**

- **AR（主动脉瓣反流）**：主动脉瓣关闭不全，舒张期主动脉压力高于左室时血流从主动脉反流入左室，产生舒张早期高调递减型吹风样杂音（200–500 Hz），紧随 S2 之后开始。重度或急性 AR 可因左室充盈骤增而出现 **S3**（S2 后约 120–180 ms，20–50 Hz），提示左室容量负荷过重。由于 AR 默认保留状态包含舒张期（keep_states = [1, 3, 4]），S3 若存在则**随舒张期自动保留**，无需额外操作。
- **MS（二尖瓣狭窄）**：二尖瓣开口面积缩小，舒张期左房向左室充盈时血流受阻，产生舒张中晚期低调隆隆样杂音（30–120 Hz）。瓣叶弹性尚存时几乎必然伴有**开瓣音（Opening Snap，OS）**（S2 后 60–120 ms，80–250 Hz），OS 是 MS 的特征性听诊体征。MS 默认保留舒张期（keep_states = [1, 3, 4]），OS **随舒张期自动保留**。注：OS 是高频骤开音，与低频充盈心音 S3（20–50 Hz）性质不同，在频谱上可区分。

**Step 2 为何同时保留 S1**：舒张期杂音（AR/MS）虽然发生在 S2 之后，但 S1 标志着本心动周期收缩期的开始，其时长决定了对应舒张期的起止位置。将 S1 纳入保留范围，可让模型理解完整的 R-R 间期节律，避免因缺少 S1 而无法正确定位舒张期的起始边界。

**额外心音汇总**：

| 疾病 | 额外心音 | 频率 | 时相 | 是否随默认掩膜保留 | 备注 |
|------|---------|------|------|:-----------------:|------|
| AS | S4 | 20–70 Hz | S1 前约 100 ms（舒张晚期） | **否** | 偶发（重度30–40%），DWT 辅助检测 |
| AR | S3 | 20–50 Hz | S2 后 120–180 ms（舒张早期） | **是** | 重度/急性可有，随舒张期自动保留 |
| MR | S3 | 20–50 Hz | S2 后 120–180 ms（舒张早期） | **否** | 偶发（中重度40–60%），DWT 辅助检测 |
| MS | OS（开瓣音） | 80–250 Hz | S2 后 60–120 ms（舒张早期） | **是** | 几乎必然存在，随舒张期自动保留；高频，非 S3/S4 |
| MVP | — | — | — | — | 无 S3/S4 |
| N | — | — | — | — | 全保留 |

## 9 已知局限与可扩展方向

| 局限 | 说明 |
|------|------|
| 模型泛化性 | Springer HSMM 仅用 5 条训练记录，对高噪声/心律不齐数据容易出错，需借助手动编辑器校正 |
| 疾病检测依赖命名 | 疾病标签从路径关键字提取，要求文件名或目录名包含 AS/AR/MR/MS/MVP/N |
| 多标签疾病 | 当前每条记录只关联一个疾病代码，不支持混合疾病 |
| 批量编辑 | 手动分段编辑器每次仅针对单个片段，大批量数据校正效率有限 |
| S3/S4 检测为启发式 | DWT 辅助检测基于固定时间窗和能量阈值，低频噪声环境下存在假阳性，检测结果仅供参考，不能替代临床判断 |

**可扩展方向**：
- 集成深度学习分割模型（如 U-Net 1D）以提升质量差数据的分割精度
- 支持批量分段校正标注模式
- 在 `segments_metadata.json` 中记录哪些片段经过了手动校正标志
- 增加频谱图（短时傅里叶变换 / Mel 谱）辅助视图
- 输出与 CirCor/PhysioNet 兼容的 `.hea` / `.tsv` 元数据格式
