# 基于 ONMF 与 CNN 的瓣膜性心脏病 PCG 信号分类系统实现报告

**参考文献**：Torre-Cruz et al., "Detection of valvular heart diseases combining orthogonal non-negative matrix factorization and convolutional neural networks in PCG signals", *Journal of Biomedical Informatics*, 145:104475, 2023.

---

## 1. 项目背景与目标

瓣膜性心脏病（Valvular Heart Disease，VHD）是全球范围内高发的心血管疾病，传统临床依赖专科医师听诊心音图（PCG）进行初步筛查，主观性强、效率偏低。Torre-Cruz et al. 2023 提出了一种两阶段自动分类框架：首先用**正交非负矩阵分解（ONMF）**从 PCG 信号中提取结构性时序特征，再送入**卷积神经网络（CNN）**完成正常/异常二分类。

本项目完整复现上述工作，并配备交互式 GUI，支持直接处理 PhysioNet 2016 Challenge 数据集（1000 条 PCG，含 5 类标签）。

**数据集类别说明**：

| 标签 | 含义 | 样本数 |
|------|------|--------|
| N（Normal） | 正常心音 | 200 |
| AS（Aortic Stenosis） | 主动脉瓣狭窄 | 200 |
| MR（Mitral Regurgitation） | 二尖瓣关闭不全 | 200 |
| MS（Mitral Stenosis） | 二尖瓣狭窄 | 200 |
| MVP（Mitral Valve Prolapse） | 二尖瓣脱垂 | 200 |

论文将后四类统一归为"异常（Abnormal）"，进行二分类（Normal vs. Abnormal）。

## 2. 第一阶段：ONMF 预处理

### 2.1 算法流程

预处理完整遵循论文 Algorithm 1，对每条 PCG 录音执行以下步骤：

```
原始 PCG 信号（任意采样率）
    ↓ 降采样至 4096 Hz
    ↓ Step 1：幅度归一化  x ← x / max|x|
    ↓ Step 2：STFT 幅度谱（Hamming 窗，N=128，hop=32，FFT=1024）
    ↓         谱最大值归一化  X ← X / max(X)
    ↓ Step 3：频域频带截取  B_A = [200, 700] Hz（论文最优参数）
    ↓ Step 4：随机正数初始化 W（频谱基底）、H（时序激活）
    ↓ Step 5–6：ONMF 乘法更新规则迭代（最多 60 次）
    ↓ Step 7：QR 正交化投影，防止 W 基底退化
    ↓
    H 矩阵（128 × T，时序激活特征）  ← 作为 CNN 输入
```

**关键参数**（论文 Table 2 / Section 4.2）：

| 参数 | 值 | 说明 |
|------|----|------|
| 目标采样率 | 4096 Hz | 所有 PCG 统一重采样 |
| STFT 窗长 | 128 样本（31.25 ms） | Hamming 窗 |
| STFT Hop | 32 样本（75% 重叠） | 时间分辨率 7.81 ms |
| FFT 点数 | 1024 | 频率分辨率 4 Hz |
| **频带（最优）** | **B_A = [200, 700] Hz** | 覆盖异常心音主能量 |
| ONMF 秩 | K = 128 | 基底数量 |
| 最大迭代 | I = 60 | 收敛判据：相对误差变化 < 1e-6 |

### 2.3 ONMF 更新规则

目标：$V \approx WH$，约束 $W \geq 0$，$H \geq 0$，$W^\top W \approx I$（W 列正交）

**H 更新**：

$$H \leftarrow H \odot \frac{W^\top V}{W^\top W H + \varepsilon}$$

**W 更新**：

$$W \leftarrow W \odot \frac{V H^\top}{W H H^\top + \varepsilon}$$

每次 W 更新后执行 QR 正交化并作非负投影；对"死亡"（全零）列回退至归一化乘法更新值，防止基底退化。

### 2.4 H 矩阵特征：核心发现

论文最重要的结论：**W 矩阵**（频域基底）对各类 PCG 几乎相同，区分度极低；而 **H 矩阵**（时序激活）差异显著——

- **正常心音（N）**：H 矩阵呈现**等间距规律竖条纹**，反映心跳周期稳定、多基底同步激活；
- **异常心音**（AS / MR / MS / MVP）：H 矩阵呈现**不规律间距或弥散激活**，反映瓣膜病变导致心动周期紊乱。

这是 H 矩阵能够区分正常与异常心音的根本原因，也解释了时序特征相比纯频域特征精度大幅提升（约 +45%）的机制。

## 3. 第二阶段：CNN 分类

### 3.1 输入格式

ONMF 输出的 H 矩阵尺寸为 $(K \times T)$，时间帧数 $T$ 随录音时长变化。为满足 CNN 固定尺寸输入要求，使用双线性插值将 H 统一调整为 $(1 \times 128 \times 256)$（单通道，归一化至 $[0, 1]$），作为 CNN 的输入图像。

### 3.2 UjaNet 架构（论文自研，Table 3）

论文提出了 UjaNet 作为专为 PCG-H 矩阵设计的 CNN，结构如下：

```
输入 (1, 128, 256)
  → Conv2D(1→16, 5×5, LeakyReLU) → MaxPool2D(2×2)   → (16, 62, 126)
  → Conv2D(16→32, 5×5, LeakyReLU) → MaxPool2D(2×2)   → (32, 29, 61)
  → Flatten → FC(→100, LeakyReLU) → Dropout(0.5)
  →           FC(→50,  LeakyReLU) → Dropout(0.5)
  →           FC(→2)
输出 logits (batch, 2)    参数量：≈ 5,679,300
```

系统同时支持 LeNet-5、AlexNet、VGG16、ResNet50、GoogLeNet 五种对比架构，全部适配单通道输入。

### 3.3 训练策略（论文 Section 4.2）

| 超参数 | 值 | 说明 |
|--------|----|------|
| 优化器 | Adam | lr = 0.001 |
| 损失函数 | CrossEntropyLoss | 对应二分类 |
| 最大轮数 | 30 epochs | 早停可提前终止 |
| Batch Size | 16 | |
| 早停 | patience = 10 | 监控 val_loss 不下降则停止 |
| Dropout | 0.5 | UjaNet FC 层正则化 |
| **评估协议** | **10-fold CV × 5 次重复 = 50 个评估点** | 生成 box plot 数据（论文 Fig. 7） |

10-fold 交叉验证数据划分方式（StratifiedShuffleSplit，保持类别比例）：

```
全量数据（1000 样本）
    ↓ 外层：75% 训练+验证 / 25% 测试（每 fold，stratified）
    ↓ 内层：75% 训练 / 25% 验证（用于早停监控）
    ↓ 共 10 fold × 5 次重复 = 50 个测试集评估结果
    → 生成 box plot 展示指标分布（复现论文 Fig. 7）
```

### 3.4 评估指标

$$\text{Sensitivity (Se)} = \frac{TP}{TP + FN} \quad \text{（异常心音检出率，越高漏报越少）}$$

$$\text{Specificity (Sp)} = \frac{TN}{TN + FP} \quad \text{（正常心音识别率，越高误报越少）}$$

$$\text{Score (Sco)} = \frac{Se + Sp}{2} \quad \text{（PhysioNet 2016 Challenge 官方排名指标）}$$

$$\text{Precision (Pre)} = \frac{TP}{TP + FP} \qquad \text{F1} = \frac{2 \cdot Pre \cdot Se}{Pre + Se}$$

## 6. 分析与讨论

### 6.1 H 矩阵作为 CNN 输入的合理性

ONMF 通过正交约束，使各列基底 $W_k$ 尽量捕获信号中相互独立的频谱模式，从而使 H 矩阵中每行 $H_k$ 代表第 $k$ 个频谱模式在时间上的激活强度。对于周期规律的正常心音，多个基底以固定时间间隔被激活，H 矩阵呈现竖条纹规律结构；瓣膜病变导致的心动周期紊乱直接破坏这种规律性，使 H 矩阵纹理紊乱，形成 CNN 可识别的视觉判别特征。

从信号处理角度看，ONMF 完成了从**时频域稀疏表示**到**视觉判别特征**的转化，CNN 再从中学习空间纹理模式，两个阶段的特性高度互补。

### 6.2 频带选择的影响

论文对比了四种频带的分类效果（Table 2）：

| 频带 | 范围 | 物理意义 |
|------|------|----------|
| B_A（**最优**） | 200–700 Hz | 覆盖大多数瓣膜杂音主频段 |
| B_C | 20–700 Hz | 正常 S1/S2 心音 + 异常杂音综合 |
| B_N | 20–200 Hz | 正常心音低频主体 |
| B_F | 20–2048 Hz | 全频带（含无关高频噪声） |

B_A 频带专注于异常心音的高频成分，有效过滤了正常心音主体能量，使 H 矩阵在正常与异常类之间差异最大化，分类性能最优。B_N 频带反之，主要捕获正常心音能量，对异常区分度低。

### 6.3 UjaNet 设计优势

UjaNet 专为 PCG 的 H 矩阵特征设计，相较于通用迁移学习架构具有以下特点：

- **参数量适中**（≈ 570 万），不易在小数据集（1000 样本）上过拟合；
- **LeakyReLU** 激活函数避免神经元死亡，适合 H 矩阵的稀疏激活特征；
- **双层 Dropout(0.5)** 与早停策略配合，有效控制泛化误差；
- 无需 ImageNet 预训练权重，从头训练即可取得竞争性结果，适合医学信号这类与自然图像差异较大的数据。

### 6.4 局限性与泛化能力讨论

尽管本次复现在 PhysioNet 2016 Challenge 数据集上取得了较高的分类指标（Sco = 0.9892），但在实际应用中仍存在以下局限，需要审慎对待：

**（1）公开数据集的信号质量偏理想**

PhysioNet 2016 数据集由多个机构标准化采集，正常与异常心音在时频域上的区别相对明显：正常心音节律规整，H 矩阵呈清晰等间距条纹；各类异常心音因瓣膜病变呈现各自特征性的紊乱纹理。这使得 ONMF + CNN 流水线能够充分发挥优势，H 矩阵的视觉判别性很强。然而，临床实际采集的 PCG 信号往往夹杂较多干扰：呼吸音叠加、体动伪影、听诊器接触噪声、体表脂肪层对高频信号的衰减等，均可能使 H 矩阵的条纹结构模糊甚至湮没，导致模型在公开数据集上学到的判别模式在现实信号中失效。此外，不同采集设备（电子听诊器 vs. 接触式传感器）的频率响应差异也会影响 B_A 频带内的能量分布，进一步加大域偏移风险。

**（2）自建数据集样本量不足，未进行验证实验**

本项目在方法复现阶段仅使用 PhysioNet 2016 公开数据集（1000 例）进行训练与评估，并未在自采临床 PCG 数据上开展验证实验。主要原因在于自建数据集目前的样本规模尚不满足 10-fold 交叉验证的统计稳健性要求——深度学习模型在小样本场景下对每个 fold 的划分方式极为敏感，少量样本同时承担训练、验证、测试三重角色，评估结果方差大、可重复性差，难以得出可靠结论。此外，自建数据的标注质量与采集标准化程度也有待进一步完善。

**（3）跨域泛化能力尚待验证**

由于上述两点原因，现有模型的跨数据集泛化能力尚属未知。模型在 PhysioNet 数据上的高性能不能直接推论为在其他采集条件下同样有效。后续若要推向实际应用，建议：一方面扩充自建数据集的采集数量并规范标注流程；另一方面在迁移学习或域适应框架下，以公开数据预训练、自建数据微调的方式提升跨域鲁棒性。

## 7. 总结

本项目完整复现了 Torre-Cruz et al. 2023 的两阶段 PCG 分类方法：

1. **ONMF 预处理阶段**严格遵循 Algorithm 1，采用论文最优参数（B_A 频带、K=128、I=60），将 PCG 信号转化为具有强判别性的 H 时序激活矩阵，正常心音的规律竖条纹与异常心音的紊乱纹理构成 CNN 的视觉输入；

2. **CNN 分类阶段**实现了论文自研的 UjaNet 及 5 种对比架构，采用 10-fold × 5 次重复交叉验证协议，生成与论文 Fig. 7 一致的 Box Plot 评估结果，同时报告 Se / Sp / Pre / Sco / F1 全套指标；

3. 系统提供完整 **GUI 界面**，支持一键完成从 `.npy` 特征加载到训练评估可视化的全流程，无需修改代码即可调整所有超参数与评估配置，也可灵活替换 CNN 架构进行对比实验。
